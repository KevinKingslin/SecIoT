# SecIoT (frontend module)

An integrated web application for performing security analysis on IoT devices

## Prerequisites
- Node Package Manager (npm)
- Angular-cli
    - To install angular-cli run `npm install @angular/cli` after installing npm

## Initialising

From the root directory, run `npm install` to install necessary packages through the npm package manager

## Development server

Run `ng serve` to create a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if any changes are made to any of the source files.
